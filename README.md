Basic Inventory System for Console

Este es un sistema de inventario hecho en Python para usar en consola.
Fue desarrollado como práctica para aprender programación desde cero.

---

¿Qué hace?

El programa permite:

- Agregar productos
- Ver la lista de productos
- Buscar productos
- Actualizar información
- Eliminar productos
- Ver estadísticas del inventario
- Guardar datos en un archivo CSV
- Cargar datos desde un archivo CSV

---

¿Cómo usarlo?

1. Abre la terminal en la carpeta del proyecto
2. Ejecuta:

python3 main.py

3. Usa el menú escribiendo el número de la opción que quieras



Archivo principal

- "main.py" → aquí está todo el código del sistema
